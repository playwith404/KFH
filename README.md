## 🚀 프로젝트 소개

> **실전처럼 연습하고, 안전하게 대응하자.**

K-Phishing Hunterz는 보이스피싱·스캠 대응 역량을 체험형으로 학습하고,
미끼 배포·신고·검증·보상까지 이어지는 흐름을 데모로 보여주는 **모바일 레이아웃 기반 웹 서비스**입니다.

이 프로젝트는 실제 앱 기획을 웹 데모로 먼저 구현한 형태이며,
React + Vite 프론트엔드와 FastAPI 백엔드를 Docker 기반으로 분리 운영합니다.

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| 명예 헌터 온보딩 | 이메일/비밀번호 회원가입, JWT 인증, 데모용 실명인증, 교육·테스트·서약·자격증 발급 흐름 제공 |
| 메인 대시보드 | 모바일 앱 형태의 홈 화면, 오늘의 미션, 진행 상태, 빠른 액션 UI 제공 |
| 미끼 배포 | 가상의 피싱 대응용 미끼 정보 생성 및 배포 이력 확인 |
| 실전형 대응 훈련 | 실제 상황처럼 구성된 시나리오 기반 학습과 대응 연습 제공 |
| 신고 및 검증 | 사용자 신고, 운영자 검토, 상태 추적, 증빙 확인 흐름 지원 |
| 포인트 및 보상 | 활동 기반 포인트 적립, 보상 목록 조회, 교환 신청 기능 제공 |
| 운영 관리 | 공지, 콘텐츠, 사용자, 신고 데이터 관리를 위한 관리자 기능 제공 |

## 📦 시작하기

### 1. 사전 준비

- Docker / Docker Compose 설치
- PostgreSQL 서버에 `khp` 데이터베이스 생성
- Reverse Proxy와 연결할 외부 Docker 네트워크 준비

```bash
docker network create khp-net
```

### 2. 환경 변수 설정

프로젝트 루트 기준으로 필요한 `.env` 파일을 구성합니다.

- 프론트엔드 포트: `3700`
- 백엔드 포트: `8700`
- Redis 포트: `10700`
- 도메인: `https://kph.pjcloud.store`
- DB 이름: `khp`

### 3. 실행

```bash
docker compose -f docker-compose.demo.yml up -d --build
```

### 4. 접속 주소

- Web: `https://kph.pjcloud.store`
- API: `https://kph.pjcloud.store/api`
- Docs: `https://kph.pjcloud.store/api/docs`
