## рҹҡҖ н”„лЎңм қнҠё мҶҢк°ң

> **"н”јмӢұмқҙ мҳЁлӢӨлҠ” л§җмқҖ л„Ҳл¬ҙ мҲҳлҸҷм ҒмқҙлӢӨ. лӮҙк°Җ н”јмӢұм—җкІҢлЎң к°„лӢӨ."**

**мјҖмқҙнҢқ н”јмӢұ н—Ңн„°мҰҲ (K-Phishing Hunterz)** лҠ” ліҙмқҙмҠӨн”јмӢұкіј мҠӨмә мқ„ **м—ӯмңјлЎң лӮҡм•„** мӮ¬кё°кҫјмқҳ м •ліҙлҘј мҲҳм§‘н•ҳкі , к·ёл“Өмқҳ лҚ°мқҙн„°лІ мқҙмҠӨлҘј мҳӨм—јмӢңмјң лІ”мЈ„ мғқнғңкі„мқҳ кІҪм ңм„ұмқ„ л¬ҙл„ҲлңЁлҰ¬лҠ” **мӢңлҜј м°ём—¬нҳ• мӮ¬мқҙлІ„ мҲҳмӮ¬ нҳ‘л Ҙ н”Ңлһ«нҸј**мһ…лӢҲлӢӨ.

кІҪм°°мІӯ мӮ¬мқҙлІ„лІ”мЈ„ мҳҲл°© кіөлӘЁм „мқ„ мң„н•ҙ кё°нҡҚлҗң ліё н”„лЎңм қнҠёлҠ”, мӢӨм ң м•ұ кё°нҡҚмқ„ **мӣ№ лҚ°лӘЁ**лЎң лЁјм Җ кө¬нҳ„н•ң нҳ•нғңмһ…лӢҲлӢӨ. React + Vite н”„лЎ нҠём—”л“ңмҷҖ FastAPI л°ұм—”л“ңлҘј Docker кё°л°ҳмңјлЎң л¶„лҰ¬ мҡҙмҳҒн•ҳл©°, лӘЁл°”мқј м•ұкіј мң мӮ¬н•ң лӢЁмқј м»¬лҹј л Ҳмқҙм•„мӣғмқ„ м ңкіөн•©лӢҲлӢӨ.

<br/>

### рҹ§җ мҷң мқҙ н”„лЎңм қнҠёмқёк°Җ?

мӮ¬кё°кҫјл“ӨмқҖ нҡЁмңЁм„ұмқ„ к·№лҸ„лЎң л”°м§ҖлҠ” м§‘лӢЁмһ…лӢҲлӢӨ. к·ёл“Өмқҙ мҲҳм§‘н•ң DBм—җ **к°Җм§ң м •ліҙк°Җ 90% мқҙмғҒ** м„һм—¬лІ„лҰ¬л©ҙ, мӮ¬кё°лҘј м№ҳкё° мң„н•ң лҰ¬мҶҢмҠӨк°Җ кІ°кіј(мҲҳмқө)ліҙлӢӨ м»Өм§ҖкІҢ лҗҳм–ҙ **лІ”мЈ„ мғқнғңкі„ мһҗмІҙк°Җ л¬ҙл„Ҳм§‘лӢҲлӢӨ**.

мқҙкІғмқҙ л°”лЎң **лҚ°мқҙн„° мҳӨм—ј(Data Poisoning) м „лһө**мһ…лӢҲлӢӨ.

## вңЁ мЈјмҡ” кё°лҠҘ

### рҹҺ–пёҸ лӘ…мҳҲ мјҖн”јн—Ң мҳЁліҙл”©
> мӢӨлӘ… мқёмҰқ вҶ’ көҗмңЎ мқҙмҲҳ(2мӢңк°„) вҶ’ лҢҖмқ‘ н…ҢмҠӨнҠё(80м җ мқҙмғҒ) вҶ’ м„ңм•Ҫм„ң м „мһҗм„ңлӘ… вҶ’ мһҗкІ©мҰқ л°ңкёү

нҡҢмӣҗк°Җмһ… нӣ„ 4лӢЁкі„ мҳЁліҙл”©мқ„ кұ°м№ҳл©ҙ **лӘ…мҳҲ мјҖн”јн—Ң(K-Phishing Hunter)** мңјлЎң мһ„лӘ…лҗҳм–ҙ к°ҖмғҒ лІҲнҳё л°Ҹ лҜёлҒј IDлҘј л°ңкёүл°ӣмҠөлӢҲлӢӨ.

### рҹӘӨ лҜёлҒј л°°нҸ¬ мӢңмҠӨн…ң
> кёүмІҳ н”јн•ҙмһҗнҳ• / лҢҖм¶ң нқ¬л§қмһҗнҳ• / нҲ¬мһҗ мһ…л¬ёмһҗнҳ• / лЎңл§ЁмҠӨ м·Ём•Ҫмһҗнҳ•

м •мғҒ кұ°лһҳмһҗк°Җ м—°лқҪн•  мқҙмң к°Җ м—ҶлҠ” кө¬мЎ°лЎң м„Өкі„лҗң н‘ңмӨҖнҷ” лҜёлҒјлҘј м»Өл®ӨлӢҲнӢ°м—җ л°°нҸ¬н•ҳм—¬, мӮ¬кё°кҫјмқҳ мһҗл°ңм Ғ м ‘к·јмқ„ мң лҸ„н•©лӢҲлӢӨ.

### рҹҺЈ AI нҺҳлҘҙмҶҢлӮҳ мӢӨмӢңк°„ лҢҖмқ‘
> м–ҙлҰ¬мҲҷн•ң л…ёмқё / IT м„ңнҲ° мӨ‘мһҘл…„ / кёүн•ң м§ҒмһҘмқё / мҲңм§„н•ң лҢҖн•ҷмғқ

мӮ¬кё° мҲҳлІ•м—җ л§һлҠ” мөңм Ғмқҳ AI нҺҳлҘҙмҶҢлӮҳк°Җ мһҗлҸҷ л§Өм№ӯлҗҳм–ҙ мӮ¬кё°кҫјкіј лҢҖнҷ”н•©лӢҲлӢӨ. н—Ңн„°лҠ” мӢӨмӢңк°„ кҙҖм „ лӘЁл“ңлЎң м°ём—¬н•ҳл©°, кі„мўҢлІҲнҳё/URL/IP л“ұмқҙ мһҗлҸҷ м¶”м¶ңлҗ©лӢҲлӢӨ.

### рҹ”Қ 3лӢЁкі„ кІҖмҰқ мӢңмҠӨн…ң
> AI 1м°Ё нҢҗм • вҶ’ н—Ңн„° 2м°Ё кІҖнҶ  вҶ’ кІҪм°° 3м°Ё мҠ№мқё

5лҢҖ мӮ¬кё° м§Җн‘ң AI л¶„м„қ, 8к°ң н•ӯлӘ© мІҙнҒ¬лҰ¬мҠӨнҠё н—Ңн„° кІҖнҶ , лӢҙлӢ№ мҲҳмӮ¬кҙҖ мөңмў… мҠ№мқёмқҳ мӮјмӨ‘ кІҖмҰқмңјлЎң **мҳӨнғҗмқ„ мөңмҶҢнҷ”**н•©лӢҲлӢӨ. лҸҷмқј мҰқкұ°м—җ лҢҖн•ҙ 2лӘ… мқҙмғҒмқҳ н—Ңн„°к°Җ лҸ…лҰҪ мӢ кі н•ҙм•ј мҲҳмӮ¬ мқҳлў°лҗҳлҠ” **көҗм°ЁкІҖмҰқ** мӢңмҠӨн…ңмқ„ м Ғмҡ©н•©лӢҲлӢӨ.

### рҹҸҶ нҸ¬мқёнҠё & л“ұкёү мӢңмҠӨн…ң
> нӣҲл Ёмғқ вҶ’ лЈЁнӮӨ вҶ’ л Ҳк·Өлҹ¬ вҶ’ м—ҳлҰ¬нҠё вҶ’ л§ҲмҠӨн„°

лҜёлҒј л°°нҸ¬(+10P)л¶Җн„° мЎ°м§Ғ кІҖкұ° кё°м—¬(+3,000P)к№Ңм§Җ нҷңлҸҷ кё°л°ҳ ліҙмғҒмқ„ м ңкіөн•©лӢҲлӢӨ. мЈјк°„/мӣ”к°„ лһӯнӮ№, м§Җм—ӯлі„ лҢҖн•ӯм „, м—°л§җ мӢңмғҒмӢқ л“ұ кІҪмҹҒ мҡ”мҶҢлЎң м§ҖмҶҚм Ғ м°ём—¬лҘј мң лҸ„н•©лӢҲлӢӨ.

### рҹ—әпёҸ лҚ°мқҙн„° мҳӨм—ј нҳ„нҷ© (Global View)
> м „көӯ м§ҖлҸ„ кё°л°ҳ мӢӨмӢңк°„ мӢңк°Ғнҷ”

мјҖн”јн—Ңл“Өмқҳ лҜёлҒјм—җ лӮҡмқё мӮ¬кё°кҫјмқҳ мң„м№ҳ(IP кё°л°ҳ) мӢӨмӢңк°„ н‘ңмӢң, м§Җм—ӯлі„ нһҲнҠёл§ө, м „мІҙ мҳӨм—јлҸ„ м§Җн‘ңлҘј лҢҖмӢңліҙл“ңлЎң м ңкіөн•©лӢҲлӢӨ.

### рҹӣЎпёҸ кІҪм°°/мҡҙмҳҒ мҪҳмҶ”
> мӢ кі  мҠ№мқё нҒҗ / лӘЁлӢҲн„°л§Ғ лҢҖмғҒ кҙҖлҰ¬ / мӮ¬мҡ©мһҗВ·мҪҳн…җмё В·ліҙмғҒ кҙҖлҰ¬ / к°җмӮ¬ лЎңк·ё

кІҪм°° мҲҳмӮ¬кҙҖкіј мҡҙмҳҒмһҗлҘј мң„н•ң м „мҡ© мҪҳмҶ”м—җм„ң 3м°Ё кІҖмҰқ мІҳлҰ¬, мҳӨнғҗ кҙҖлҰ¬, көҗмңЎ мҪҳн…җмё  кҙҖлҰ¬ л“ұмқ„ мҲҳн–үн•©лӢҲлӢӨ.
